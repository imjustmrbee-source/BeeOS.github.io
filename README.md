📱 Virtual Phone OS (HTML / CSS / JS)

A lightweight virtual phone interface built using standard web technologies.
This project runs entirely in the browser and can be installed on mobile devices like an app using Add to Home Screen.

It is designed to simulate a phone operating system UI, launcher, or app environment.

🚀 Features
Built with HTML, CSS, and JavaScript
Works on desktop and mobile browsers
Installable on iPhone and Android
Full-screen app-like experience
Offline support (optional)
No App Store required
Easy to customize and expand
